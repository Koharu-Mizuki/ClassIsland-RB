# OCR 课表导入插件

通过拍照或截图，使用 AI 视觉识别自动导入课表和时间表。

## 功能

- 上传课表图片（PNG / JPG / WEBP）
- 调用 AI 接口自动识别时间段和科目
- 识别结果可手动编辑修正
- 支持合并到当前档案或创建新档案

## 支持的 AI 服务

| 服务商 | 推荐模型 |
|--------|----------|
| OpenAI | `gpt-4o` |
| Anthropic | `claude-opus-4-5-20251001` |
| 自定义（OpenAI 兼容接口） | 取决于服务商 |

## 使用方法

1. 在「插件设置 → OCR 课表导入」中填写 API 密钥和模型
2. 在「档案 → 导入 → 从图片导入（AI 识别）」中选择课表图片
3. 点击「开始识别」，等待 AI 返回结果
4. 检查并修正识别结果
5. 选择导入目标后点击「导入课表」

## 注意

- 识别时图片会发送至您选择的 AI 服务商，请确认课表图片不含敏感信息
- API 密钥仅保存在本地，不会上传到任何其他服务器
